🎯 Objetivo

Nesse projeto vamos mostrar que a casa sempre vence no longo prazo através de conceitos como a Lei dos Grandes Números e o house edge (vantagem matemática da casa).
📚 Conteúdo Principal
Aspecto	Descrição
Foco - Jogos de azar e as estratégias dos cassinos para maximizar lucros
Público-alvo - Pessoas interessadas em entender a matemática por trás dos cassinos
Estrutura - Múltiplas páginas com progressão de conteúdo (primeira página é introdutória)
